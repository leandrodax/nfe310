Opção de Inclusão de novo produto para o Emitente atual.


Pré-condição: Um emitente deverá estar previamente iniciado.
  1. Acessar o menu: Cadastro -> Produto

  2. Na tela de pesquisa de Produtos Cadastrados, acessar a opção Incluir

  3. Na tela de Inclusão de Produtos, preencher corretamente os campos do produto

  4. Após o preenchimento dos campos corretamente, selecionar a opção Incluir
