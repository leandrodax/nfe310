Cadastro de Produtos

O Cadastro de Produtos é uma opção para facilitar o preenchimento das Notas Fiscais eletrônicas.
Durante a criação da NF-e, o usuário poderá carregar as informações do produto previamente cadastrado na Nota Fiscal.

Este cadastro permite a inserção, alteração e exclusão e pesquisa dos produtos do Emitente.

Acesso pelo Menu: Cadastro -> Produto





Opções:

  • Pesquisa de produto

  • Inclusão de produto

  • Detalhamento/Edição de produto

  • Exclusão de produto
