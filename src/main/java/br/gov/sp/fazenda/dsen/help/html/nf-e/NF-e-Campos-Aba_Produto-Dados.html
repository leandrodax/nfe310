Produtos e Serviços - Dados


Alguns dos dados do Produto poderão ser carregados do Cadastro de Produtos  mediante a Pesquisa. Para isso, clicar em Pesquisar, realizar a busca pelo produto desejado e clicar em Carregar. Os dados contidos no Cadastro de produto serão inseridos nos campos de produto. Além disso, ao selecionar o tributo, caso este esteja cadastrado no produto, também será carregado nos dados da NF-e.
