Cadastro de Produtos

Detalhamento/Edição de produto cadastrado

Opção de detalhamento;edição de um produto cadastrado do Emitente atual.


Pré-condição: Um emitente deverá estar previamente iniciado.
  1. Acessar o menu: Cadastro -> Produto

  2. Realizar a pesquisa pelo produto cadastrado que será editado

  3. Selecionar o produto a ser editado;

  4. Após a seleção do produto, a opção de Detalhar ficará disponível. Selecionar a opção Detalhar

  5. A tela de detalhamento com os dados do produto selecionado serão apresentados. Selecionar a opção Editar para a edição dos campos do produto

  6. Após finalizar a edição dos campos, clicar em Alterar para salvar as modificações
