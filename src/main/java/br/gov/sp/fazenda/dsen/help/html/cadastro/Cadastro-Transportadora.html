Cadastro de Transportadoras

O Cadastro de Transportadora é uma opção para facilitar o preenchimento das Notas Fiscais eletrônicas.
Durante a criação da NF-e, o usuário poderá carregar as informações da transportadora previamente cadastrada na Nota Fiscal.

Este cadastro permite a inserção, alteração e exclusão e pesquisa das transportadoras utilizadas pelo Emitente.

Acesso pelo Menu: Cadastro -> Transportadora





Opções:

  • Pesquisa de transportadora

  • Inclusão de transportadora

  • Detalhamento/Edição de transportadora

  • Exclusão de transportadora



Relacionados:
