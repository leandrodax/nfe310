Preenchimento dos quadros para inclusão e edição dos dados:

Fornecimentos diários



1. No quadro Fornecimentos Diários Clicar em Incluir;
2. Digitar o dia do mês e a quantidade fornecida naquele dia, clicar em OK;
3. A edição dos dados pode ser realizada utilizando o mesmo procedimento com o botão Detalhar. A exclusão dos dados poderá ser realizada utilizando-se o botão Excluir.

Dedução



1. No quadro Dedução, clicar em Incluir;
2. Digitar o valor da dedução e a descrição do motivo da dedução clicar em OK;
3. A edição dos dados pode ser realizada utilizando o mesmo procedimento com o botão Detalhar.A exclusão dos dados poderá ser realizada utilizando-se o botão Excluir.
