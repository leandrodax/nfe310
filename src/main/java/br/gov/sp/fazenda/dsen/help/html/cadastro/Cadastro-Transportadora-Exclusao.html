Cadastro de Transportadoras

Exclusão de transportadora cadastrada

Opção de exclusão de uma ou mais transportadoras cadastradas do Emitente atual.


Pré-condição: Um emitente deverá estar previamente iniciado.


Existem duas formas de exclusão de transportadora:

1 ) Pela listagem de transportadoras cadastradas
  1. Acessar o menu: Cadastro -> Transportadora

  2. Realizar a pesquisa pela(s) transportadora(s) cadastrada(s) que será(ão) excluída(s);

  3. Selecionar a(s) transportadoras) a ser(em) excluída(s);

  4. Selecionar a opção Excluir

  5. Confirmar a exclusão


2 ) Pela tela de detalhe da transportadora
  1. Na tela de detalhe dos dados da transportadora, selecionar a opção Excluir para excluir a transportadora selecionada

  2. Confirmar a exclusão
