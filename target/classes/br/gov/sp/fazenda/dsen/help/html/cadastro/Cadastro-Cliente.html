Cadastro de Clientes

O Cadastro de Clientes é uma opção para facilitar o preenchimento das Notas Fiscais eletrônicas.
Durante a criação da NF-e, o usuário poderá carregar as informações do cliente previamente cadastrado na Nota Fiscal.

Este cadastro permite a inserção, alteração e exclusão e pesquisa dos clientes do Emitente.

Acesso pelo Menu: Cadastro -> Cliente





Opções:

  • Pesquisa de cliente

  • Inclusão de cliente

  • Detalhamento/Edição de cliente

  • Exclusão de cliente



Relacionados:
  • Preenchimento dos campos de cliente
