Pesquisa de produtos cadastrados

Opção para a pesquisa dos produtos cadastrados do Emitente atual.


Pré-condição: Um emitente deverá estar previamente iniciado.
  1. Acessar o menu: Cadastro -> Produto

  2. Na tela de pesquisa de Produtos Cadastrados, preenher com o filtro de pesquisa de produto desejado.

  3. Clicar em Pesquisar

  4. Para limpar os campos do filtro, clicar em Limpar Filtro



Filtro de Pesquisa de Produto:




Código Código do Produto
Preenchimento de até 60 caracteres;
A pesquisa retornará todos os produtos cujo código inicie com os caracteres digitados
Descrição Preenchimento de até 60 caracteres;
A pesquisa retornará todos os produtos cuja Descrição contenha a expressão digitada


Observações:
  1. Os campos do filtro de pesquisa somam-se na busca: caso seja preenchido o campo Código e o campo Descrição, a busca utilizará os dois campos para realizar a pesquisa

  2. Caso não sejam preenchidos dados nos campos do filtro, a pesquisa retornará TODOS os produtos cadastrados do emitente atual
