Pesquisa de transportadoras cadastradas

Opção para a pesquisa das transportadoras cadastradas do Emitente atual.


Pré-condição: Um emitente deverá estar previamente iniciado.
  1. Acessar o menu: Cadastro -> Transportadora

  2. Na tela de pesquisa de Transportadoras Cadastradas, preenher com o filtro de pesquisa de transportadora desejado.

  3. Clicar em Pesquisar

  4. Para limpar os campos do filtro, clicar em Limpar Filtro



Filtro de Pesquisa de Transportadora:




CNPJ/CPF Preenchimento de até 14 dígitos;
A pesquisa retornará todas as transportadoras cujo CNPJ ou CPF inicie com o número digitado
Inscrição Estadual Preenchimento de até 14 dítigos;
A pesquisa retornará todas as transportadoras cuja IE inicie com o número digitado
Nome/Razão Social Preenchimento de até 60 caracteres;
A pesquisa retornará todas as transportadoras cujo Nome ou  Razão Social contenha a expressão digitada


Observações:
  1. Os campos do filtro de pesquisa somam-se na busca: caso seja preenchido o campo CNPJ/CPF e o campo Inscrição Estadual, a busca utilizará os dois campos para realizar a pesquisa; caso seja preenchido o campo CNPJ/CPF, o campo Inscrição Estadual e o campo Nome/Razão Social, a busca utilizará os três campos para realizar a pesquisa

  2. Caso não sejam preenchidos dados nos campos do filtro, a pesquisa retornará TODAS as transportadoras cadastradas do emitente atual
